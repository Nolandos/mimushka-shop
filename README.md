#Mimushka - shop app
MERN app for online shop with client and admin layout.

#Demo : 
App was deployed on heroku server:

https://mimushka-shop.herokuapp.com/

#Technologies :
- React + Redux + Hooks
- Node + express + mongodb + mongo Atlas
- Formik with Yup validation
- CKEditor
- Sass
- Axios
- Nodemailer
- Passport 
- and other...
